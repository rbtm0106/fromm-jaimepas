# fromm-jaimepas

simple fromm VOD proxy server

## Requirements

You need Python and the following dependencies:

* Flask
* Requests
* PyCryptodome

## Installation

```bash
git clone https://github.com/rbtm0106/fromm-jaimepas.git
cd fromm-jaimepas

pip install -r requirement.txt
```

## Run the app

```bash
python app.py
```


## Tested in this environment

* Windows 11
* Python 3.13
* firefox 145.0.1
