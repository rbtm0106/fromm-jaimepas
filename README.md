# fromm-jaimepas

simple fromm VOD proxy server

## Requirements

You need Python and the following dependencies:

* flask
* PyCryptodome
* requests

## Installation

```bash
git clone https://github.com/rbtm0106/fromm-jaimepas.git
cd fromm-jaimepas

pip install -r requirement.txt
```

## Run the app

```bash
python app.py
```
