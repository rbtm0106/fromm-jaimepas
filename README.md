# fromm-jaimepas

simple fromm VOD proxy server

## Requirements

You need Python 3.13 and the following dependencies:

* Flask
* Requests

## Tested environment

* Windows 11
* Python 3.13

## Installation

```bash
git clone https://github.com/rbtm0106/fromm-jaimepas.git
cd fromm-jaimepas

pip install -r requirement.txt
```

## Run the app

```bash
python app.py
```
